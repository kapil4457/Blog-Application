# Blog-Application

## Figma Design
https://www.figma.com/file/zIPVbMfLQtr0WRlM9acmeE/for-blogs-(Community)?node-id=0%3A1&t=JNbsN2yMzDATKiOW-0

## Features : 
 ### Author Side
 1) Create Post
 2) Edit Post
 3) Post Analysis
 4) Delete Post
 5) Manage NewsLetter Subscription
 
 ### Reader Side
 1) Read post
 2) Search for a particular post
 3) Filter based on category , Author 
 4) Like a post
 5) share a post

## Technology used
1) React.js
2) MongoDb
3) Node.js
4) Express.js
5) Material UI
6) Framer Motion
